🛒 Online Shopping Portal
An interactive E-commerce platform that provides a seamless shopping experience for users. This project demonstrates full-stack development capabilities, focusing on inventory management, user cart logic, and a clean UI.

🌟 Features
Dynamic Product Catalog: View products by categories with detailed descriptions.

Shopping Cart: Add, remove, and update quantities of items dynamically.

User Management: Secure user registration, login, and profile updates.

Search & Filters: Quickly find products using the search bar or category filters.

Order Tracking: Check the status and history of placed orders.

Admin Panel: Comprehensive control for admins to manage products, categories, and users.

🛠️ Technologies Used
Frontend: HTML5, CSS3, JavaScript (Bootstrap Framework)

Backend: PHP / Laravel (Update based on your current implementation)

Database: MySQL

Package Manager: Composer (if using Laravel)

⚙️ How to run the Online Shopping Portal using PHP and MySQL

1. Download the  zip file

2. Extract the file and copy shopping folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name shopping

6. Import shopping file(given inside the zip package in SQL file folder)

7.Run the script http://localhost/shopping (frontend)

8. Link for admin Panel : http://localhost/shopping/admin
